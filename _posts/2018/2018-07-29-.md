I would use this:

``` 
[[ $(Command1) == 1234 ]] && Command2
```

- The `[[` and `]]` tests if the condition inside is true
- `&&` executes `Command2` if the condition tested true
- `$(...)` executes command inside parenthesis and returns output (what was echoed inside `Command1`
