Instead of using:

``` 
$ resize 40 80
resize: Can't set window size under VT100 emulation
```

Try using:

``` 
printf '\e[8;40;80t'
```

You can also use `Edit` -> `Profile preferences` and set the initial screen size every time your terminal opens.
